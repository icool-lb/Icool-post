# تطبيق iCOOL Photo Branding - النسخة v14 الاحترافية

هذه النسخة مبنية على واجهة v13 نفسها مع إضافات احترافية جديدة.

## ما الذي أُضيف
- رفع لوغو مخصص.
- إمكانية الرجوع إلى لوغو iCOOL الافتراضي.
- تغيير ألوان الفوتر:
  - الأزرق الأساسي
  - الأزرق الداكن
  - البرتقالي
  - لون النص والأيقونات
- الإبقاء على التحكم بمحتوى الفوتر:
  - الموقع
  - الهاتف
  - الخدمات
- الإبقاء على اختيار الإطار قبل التوليد.
- الإبقاء على الزوم وتحريك الصورة داخل الإطار.
- الإبقاء على تحريك اللوغو والسلوغان وخيارات الخط والخلفية الضبابية.

## التشغيل محلياً
```bash
npm install
npm run dev
```
